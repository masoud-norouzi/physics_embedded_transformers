\section{Testing and Execution}

\subsection{Test Suite}

The test suite covers:

\begin{itemize}
  \item configuration-to-device resolution;
  \item centerline ordering, orientation, arc length, tangent, and normal behavior;
  \item device region-label construction and connectivity;
  \item droplet ellipse rasterization and occupancy normalization;
  \item baseline branch-hydraulic equations and flow conservation;
  \item centered finite-difference velocity validation;
  \item full-device CFD geometry, mesh generation, solver assembly, and
        field-level mesh verification.
\end{itemize}

Run all tests with:

\begin{verbatim}
.\.venv\Scripts\python.exe -m pytest tests
\end{verbatim}

The latest validation before this documentation update passed:

\begin{verbatim}
143 passed
\end{verbatim}

\subsection{Primary Commands}

\begin{enumerate}[label=\arabic*.]
  \item Build calibrated device geometry:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.geometry.build_geometry \
  --experiment configs\experiments\video_2.yml \
  --overwrite
\end{verbatim}

  \item Build device region labels:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.geometry.build_device_regions \
  --experiment configs\experiments\video_2.yml \
  --frame-index 0 \
  --overwrite
\end{verbatim}

  \item Compute droplet occupancy:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.occupancy.compute_droplet_occupancy \
  --experiment configs\experiments\video_2.yml \
  --overwrite
\end{verbatim}

  \item Compute baseline hydraulics:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.physics.compute_baseline_hydraulics \
  --experiment configs\experiments\video_2.yml \
  --overwrite
\end{verbatim}

  \item Run branch-velocity validation:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.validation.validate_droplet_branch_velocity \
  --experiment configs\experiments\video_2.yml \
  --overwrite
\end{verbatim}

  \item Build full-device CFD mesh:
\begin{verbatim}
.\.venv\Scripts\python.exe -m scripts.physics.build_full_device_mesh
\end{verbatim}

  \item Run full-device scalar mesh convergence:
\begin{verbatim}
$env:MPLCONFIGDIR='outputs\.matplotlib'
.\.venv\Scripts\python.exe scripts\physics\run_full_device_convergence.py `
  --target-sizes-um 48 36 24 18 12 `
  --output-dir outputs\physics\full_device_cfd\convergence_alpha0_fine
\end{verbatim}

  \item Verify the frozen \(24~\mu\mathrm{m}\) full-device field:
\begin{verbatim}
$env:MPLCONFIGDIR='outputs\.matplotlib'
.\.venv\Scripts\python.exe scripts\physics\verify_full_device_field_convergence.py `
  --grid-spacing-um 6 `
  --output-dir outputs\physics\full_device_cfd\convergence\field_comparison_24_vs_12
\end{verbatim}
\end{enumerate}

\subsection{Current Scope Boundaries}

The project currently does not implement:

\begin{itemize}
  \item transformer training;
  \item hydraulic parameter fitting;
  \item droplet slip modeling;
  \item junction loss modeling.
\end{itemize}
