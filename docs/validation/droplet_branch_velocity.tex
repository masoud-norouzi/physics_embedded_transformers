\section{Validation: Droplet Speed versus Branch Velocity}

\subsection{Files}

\begin{itemize}
  \item \texttt{scripts/validation/validate\_droplet\_branch\_velocity.py}
  \item \texttt{tests/test\_droplet\_branch\_velocity\_validation.py}
\end{itemize}

\subsection{Scientific Question}

This validation asks whether temporal variations in measured droplet speed
co-vary with temporal variations in the calculated baseline branch superficial
velocity. It is not a proof that droplet speed equals branch velocity.

The measured droplet speed is derived from tracked centroid positions in
\texttt{px/frame}. The baseline branch velocity is in
\(\mu\mathrm{m}/\mathrm{s}\). Because the comparison uses per-track
standardization, it compares temporal variation, not absolute velocity scale.

\subsection{Measured Speed from Centered Difference}

For each track, rows are sorted by frame. For interior frames with adjacent
observations exactly one frame apart:

\[
v_x(t)
=
\frac{x(t+1)-x(t-1)}{2},
\qquad
v_y(t)
=
\frac{y(t+1)-y(t-1)}{2}.
\]

The measured speed is:

\[
u_{\mathrm{drop}}(t)
=
\sqrt{v_x(t)^2+v_y(t)^2}.
\]

Endpoints and samples adjacent to tracking gaps are excluded. Track boundaries
are never mixed.

\subsection{Branch-Interior Filtering}

Samples are retained only when a droplet is clearly inside exactly one branch:

\[
w_{\mathrm{left}} \ge 0.95
\quad \mathrm{xor} \quad
w_{\mathrm{right}} \ge 0.95.
\]

For left-branch samples:

\[
u_{\mathrm{branch}}(t) = U_{\mathrm{left}}(t).
\]

For right-branch samples:

\[
u_{\mathrm{branch}}(t) = U_{\mathrm{right}}(t).
\]

There is no interpolation between left and right branch velocities.

\subsection{Normalization and Correlation}

For each selected \((\texttt{track\_id}, \texttt{branch})\) segment, both
signals are standardized independently:

\[
z_{\mathrm{drop}}(t)
=
\frac{u_{\mathrm{drop}}(t)-\bar{u}_{\mathrm{drop}}}
{\mathrm{std}(u_{\mathrm{drop}})},
\]

\[
z_{\mathrm{branch}}(t)
=
\frac{u_{\mathrm{branch}}(t)-\bar{u}_{\mathrm{branch}}}
{\mathrm{std}(u_{\mathrm{branch}})}.
\]

The standard deviation uses \(ddof=1\). Pearson and Spearman correlations are
computed at zero lag on the final aligned normalized samples. No smoothing or
lag optimization is applied.

\subsection{Candidate Selection}

The current data contain shorter right-branch interior segments than left-branch
segments. The selected cutoff is:

\[
n_{\min}=60.
\]

This prevents the entire right branch from being excluded. Candidate ranking is
deterministic and based on branch-velocity standard deviation, not correlation.
When both branches have enough candidates, the validator selects five left and
five right examples.
